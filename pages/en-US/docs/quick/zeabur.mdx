## How to Use

- You can access your archnet server for initial configuration by visiting `server-domain.zeabur.app`.
- You can access your archnet admin panel for management operations by visiting `admin-domain.zeabur.app`.
- You can access your archnet user panel for usage operations by visiting `user-domain.zeabur.app`.

## Notes

- After updating the server domain, admin domain, and user domain, please promptly update the corresponding environment variables and restart the service.

## Additional Information

- For environment variable configuration, please refer to the [ArchNet Official Documentation](https://archnet.dev/)
