## Last ned koden

Hent koden fra det offisielle GitHub-repositoriet:

```bash
# Last ned den nyeste versjonen av koden
curl -LO https://github.com/ArchNets/frontend/releases/download/v1.0.0/archnet-admin-web.tar.gz

# Pakk ut filen
tar -xzvf archnet-admin-web.tar.gz

# Gå inn i kodekatalogen
cd archnet-admin-web
```

## Konfigurer miljøvariabler

Opprett `apps/admin/.env`-filen og legg til nødvendige miljøvariabler (se ovenfor for konfigurasjon av miljøvariabler).

## Konfigurer PM2

Opprett `ecosystem.config.js`-filen med følgende innhold:

```javascript
module.exports = {
  apps: [
    {
      name: 'archnet-admin-web',
      script: 'apps/admin/server.js',
      interpreter: 'bun', // kan endres til node
      watch: true,
      instances: 'max',
      exec_mode: 'cluster',
      env: {
        PORT: 3000,
      },
    },
  ],
};
```

## Bruk PM2 for å starte tjenesten

```bash
pm2 start ecosystem.config.js
pm2 save
pm2 startup
```

## Bruk Node.js eller Bun for å starte tjenesten

- **Start med Node.js**:

  ```bash
  node apps/admin/server.js
  ```

- **Start med Bun**:

  ```bash
  bun apps/admin/server.js
  ```

---

## **4. Verifiser distribusjonen**

### **4.1 PM2 tjenestestatus**

Kjør følgende kommando for å sjekke:

```bash
pm2 list
```

### **4.2 Tilgang via nettleser**

Besøk `http://<din server IP>:3000` for å verifisere at tjenesten kjører.

---

## **5. Tjenestestyring**

### **PM2 kommandoer**

- Stopp tjenesten:

  ```bash
  pm2 stop archnet-admin-web
  ```

- Start tjenesten på nytt:

  ```bash
  pm2 restart archnet-admin-web
  ```

- Slett tjenesten:

  ```bash
  pm2 delete archnet-admin-web
  ```
