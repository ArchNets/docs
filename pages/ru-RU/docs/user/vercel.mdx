## Развертывание с помощью Vercel

Нажмите на кнопку ниже для быстрого развертывания:

[![Развернуть на Vercel](https://vercel.com/button)](https://vercel.com/new/clone?demo-description=ArchNet%20это%20чистый%2C%20профессиональный%2C%20и%20совершенный%20инструмент%20открытого%20исходного%20кода%20для%20прокси-панели%2C%20предназначенный%20для%20того%2C%20чтобы%20стать%20вашим%20идеальным%20выбором%20для%20обучения%20и%20практического%20использования&demo-image=https%3A%2F%2Furlscan.io%2Fliveshot%2F%3Fwidth%3D1920%26height%3D1080%26url%3Dhttps%3A%2F%2Fuser.archnet.dev&demo-title=Пользовательский%20интерфейс%20ArchNet&demo-url=https%3A%2F%2Fuser.archnet.dev%2F&from=.&project-name=archnet-user-web&repository-name=archnet-web&repository-url=https%3A%2F%2Fgithub.com%2FArchNets%2Farchnet-web&root-directory=apps%2Fuser&skippable-integrations=1)

**Шаги:**

1. Войдите или зарегистрируйтесь на [Vercel](https://vercel.com/) .
2. Нажмите кнопку развертывания, создайте форк репозитория и выберите `apps/user`.
3. Настройте переменные окружения.
4. Нажмите **Deploy**, чтобы начать развертывание.
